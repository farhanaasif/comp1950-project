Project: BCIT COMP1950

Our development approach to this project

Our Mission

We wanted to revamped Comp-1950 with rich HTML5 and CSS styles We had great ideas about making it appealing and more useful but due to time constraints, we were not able to implement all the the ideas we had. However, we used online available tools to fast track the process.


Introduction page is the first visited page to clients. We created the layout to impress our professional design abilities and all navigations are simple to easy access for users.


We split the whole project to two teams:


TEAM 1 - Eunhey and Orkideh

They worked on part 1 of the project that included Introduction, Sitemap and Rational pages of the team website. This team aimed to use build a professional looking website to promote our project.



TEAM 2 - Farhana and Sara

This team was mainly focused on analyzing existing website, and propose a new solution that would mainly use fancy CSS3 and HTML5 techniques to make the website more intuitive and attractive. We called our proposed solution "protosite". This team mainly worked on two seperate yet coherent layouts. Using those layout following pages:


  • Home Page
  • Lecture 1
  • Lecture 2
  • Assignments
